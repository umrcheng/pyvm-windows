# 😶‍🌫️使用教程

1. 下载软件包, 到指定目录
  [下载地址](https://github.com/umrcheng/pyvm-windows/releases)

2. 下载完成后解压, 然后复制路径添加进环境变量

3. 打开新终端输入 `pyvm -v`, 输出如下表示安装成功

```bash
C:\Users\umrcheng>pyvm -v
pyvm version 1.1.0
```

4. 具体命令如下
```bash
C:\Users\umrcheng>pyvm -h
usage: pyvm.exe [-h] [--version] {use,ls,ls-av,list,list-available,install,uninstall} ...

Python版本管理器

positional arguments:
  {use,ls,ls-av,list,list-available,install,uninstall}
                        命令列表
    use                 [version] 使用指定版本的Python
    ls                  列出已安装的Python版本
    ls-av               列出可用的Python版本
    list                列出已安装的Python版本
    list-available      列出可用的Python版本
    install             [version] 安装指定版本的Python
    uninstall           [version] 卸载指定版本的Python

options:
  -h, --help            show this help message and exit
  --version, -v         显示当前版本信息
```
